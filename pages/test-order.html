Crear Pedido de Prueba

Modo de Prueba: Este botón crea una transacción de prueba en la base de datos sin usar dinero real. Ideal para testing.

¿Qué hace este botón?

  • NO involucra dinero real
  • Aparecerá en "Mis Compras" inmediatamente
  • Estado: Aprobado automáticamente
